## ğŸ§‘â€ğŸ’» How It Works

-   The UI views (Login & Signup) are dynamically rendered using
    TypeScript functions.
-   DOM elements are strongly typed using:
    -   `HTMLElement`
    -   `HTMLInputElement`
-   Password visibility is toggled by switching input type between
    `password` and `text`.
-   Navigation between Login and Signup happens without page reload.
-   Background bubbles are generated dynamically using JavaScript logic
    compiled from TypeScript.
